# Event-Driven Order and Loyalty Platform

## Overview
A backend system built using Java 17, Spring Boot, Kafka, and Docker.

## Architecture
- API Gateway
- Auth Service (JWT)
- Order Service
- Payment Service
- Inventory Service
- Loyalty Service
- Notification Service

## Tech Stack
- Java 17
- Spring Boot 3.x
- Spring Cloud Gateway
- Apache Kafka
- PostgreSQL
- Docker & Docker Compose

## Key Features
- Event-driven architecture
- Asynchronous processing with Kafka
- Retry and Dead Letter Topics
- JWT-based security
- Distributed tracing and observability
